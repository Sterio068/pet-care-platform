## 前言：選狗飼料，不是看「肉排第幾個」就結束

狗飼料的成分表很容易讓人焦慮：肉粉是不是不好？無穀是不是比較天然？蛋白質越高越好嗎？真正要看的不是單一成分，而是這包飼料能不能長期、穩定、完整地餵給你家狗。

比較合理的選法，是同時看營養完整性、製造與品管、適不適合狗的年齡和健康狀況，再看價格和購買穩定度。

## 先看這 5 件事

| 檢查項目 | 為什麼重要 |
|----------|------------|
| 是否標示完整均衡 | 確認能不能當主食長期餵 |
| 適用生命階段 | 幼犬、成犬、老犬需求不同 |
| 熱量標示 | 方便計算每日份量 |
| 製造商與客服資訊 | 有問題時能追溯與詢問 |
| 品管與營養團隊 | 比漂亮行銷文案更重要 |

如果包裝只強調「天然、頂級、無添加」，卻沒有清楚的營養標示和客服資訊，要保守一點。

## 成分表怎麼看？

成分表依重量排序，但「重量」包含水分。新鮮肉類水分高，排在前面不代表乾物質蛋白一定最高；肉粉水分低，名稱聽起來沒那麼漂亮，但不等於一定差。

你可以用這個順序判斷：

1. 主要蛋白來源是否清楚。
2. 是否符合狗的生命階段。
3. 是否有完整營養聲明。
4. 是否標示熱量。
5. 狗吃了之後體重、糞便、皮膚和精神是否穩定。

不要只用「前三項成分」決定好壞。那只是其中一個線索。

## 常見迷思

### 迷思 1：無穀一定比較好

無穀不等於低敏，也不等於比較健康。真正食物過敏需要透過獸醫規劃排除飲食試驗，不是直接把穀物換掉。多數狗可以消化適量穀物；問題通常在整體配方是否完整均衡。

### 迷思 2：蛋白質越高越好

幼犬、工作犬、懷孕哺乳犬需求較高，但一般成犬不一定需要超高蛋白。高蛋白也不自動代表高品質，還要看消化率、胺基酸平衡和整體熱量。

### 迷思 3：貴的一定比較安全

價格會受到品牌、進口、包裝、通路和行銷影響。貴可以代表更好的原料或品管，也可能只是定位。你需要看的是可查證資訊，而不是只看每公斤價格。

### 迷思 4：鮮食一定比飼料好

鮮食可以是很好的加菜方式，但若要當長期主食，需要營養完整配方。只用雞胸肉、地瓜、花椰菜拼湊，很容易缺鈣、微量元素或脂肪酸。

## 乾糧、濕食、鮮食怎麼選？

| 類型 | 優點 | 注意事項 |
|------|------|----------|
| 乾糧 | 保存方便、成本較低、熱量穩定 | 水分低，要注意喝水與份量 |
| 濕食 | 水分高、適口性好 | 成本較高，開封後需冷藏 |
| 鮮食 | 食材可控、接受度常較好 | 長期主食需營養師或獸醫配方 |
| 處方飼料 | 針對疾病管理 | 不應自行購買替代診療 |

混餵可以，但要算總熱量。很多狗變胖不是飼料本身錯，而是主食、罐頭、零食和桌邊食物全部加起來超量。

## 換飼料的 7-10 天做法

| 天數 | 比例 |
|------|------|
| 第 1-3 天 | 25% 新飼料 + 75% 舊飼料 |
| 第 4-6 天 | 50% 新飼料 + 50% 舊飼料 |
| 第 7-9 天 | 75% 新飼料 + 25% 舊飼料 |
| 第 10 天後 | 100% 新飼料 |

若出現軟便、嘔吐、搔癢或食慾下降，先退回上一階段。症狀明顯或持續，請停下來問獸醫。

## 不同狗狗的選擇重點

| 狗狗狀況 | 選擇重點 |
|----------|----------|
| 幼犬 | 成長期完整均衡、鈣磷比例適合 |
| 結紮成犬 | 控制熱量，避免照舊餵過量 |
| 老犬 | 注意肌肉量、牙齒、腎臟與慢性病 |
| 容易胖 | 高纖、較低熱量密度，搭配份量控制 |
| 皮膚反覆癢 | 先排除跳蚤、感染，再談食物試驗 |
| 腎臟或心臟病 | 使用獸醫建議的處方或治療飲食 |

## 保存也會影響品質

1. 開封後保留原包裝，整包放進密封桶，不要直接倒入桶中。
2. 存放陰涼乾燥處。
3. 每次取飼料用乾淨量杯。
4. 不混新舊批號，方便追蹤問題。
5. 若聞到油耗味、發霉或狗吃了異常，停止餵食並保留包裝批號。
