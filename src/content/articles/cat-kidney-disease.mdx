## 前言：貓腎病最麻煩的地方，是早期很安靜

慢性腎病（CKD）是高齡貓常見的慢性疾病。它通常不是某一天突然發生，而是腎功能逐步下降。早期貓可能看起來只是喝水變多、尿塊變大、食慾忽好忽壞；等到明顯消瘦、常吐、不吃，往往已經需要更積極的醫療管理。

這篇的目的不是讓你在家診斷，而是知道哪些變化值得提早檢查，以及確診後飲食、回診和居家觀察該怎麼配合。

## 慢性腎病是什麼？

腎臟負責過濾代謝廢物、調節水分和電解質，也和血壓、紅血球生成有關。慢性腎病指的是腎功能持續下降，而且變化不是短暫脫水或急性中毒造成。

貓咪有很強的忍耐力。早期腎病不一定有明顯症狀，因此高齡貓定期健檢比等症狀出現更重要。

## 7 個早期警訊

| 警訊 | 飼主會看到什麼 |
|------|----------------|
| 多喝水 | 水碗下降變快，常去喝水 |
| 多尿 | 貓砂尿塊變大、變重 |
| 體重下降 | 背脊、髖骨變明顯 |
| 食慾不穩 | 不是完全不吃，而是吃得零碎 |
| 嘔吐變頻繁 | 不只是偶爾吐毛球 |
| 毛況變差 | 毛束感、油膩、少梳理 |
| 口氣變重 | 可能伴隨脫水、尿毒或牙齒問題 |

這些症狀不只腎病會造成。甲狀腺亢進、糖尿病、牙周病、腸胃疾病也可能類似，所以不能只看症狀下結論。

## 確診會檢查哪些項目？

| 檢查 | 目的 |
|------|------|
| 血液 creatinine、BUN | 評估代謝廢物是否累積 |
| SDMA | 輔助評估腎功能變化 |
| 尿比重 USG | 看腎臟濃縮尿液能力 |
| 尿蛋白 UPC | 評估是否有蛋白尿 |
| 血壓 | 高血壓常與腎病互相影響 |
| 電解質、磷、鉀 | 決定飲食與治療策略 |

IRIS 分期會綜合 creatinine、SDMA、尿蛋白、血壓等資訊。重點不是只知道「第幾期」，而是用分期安排追蹤頻率、飲食和用藥。

## 飲食管理：不要自己亂降蛋白

腎病飲食通常會控制磷、鈉，並調整蛋白質品質與含量。這不等於所有老貓都該吃低蛋白，也不代表越早自行換處方糧越好。錯誤飲食可能造成食慾下降或肌肉流失。

比較合理的順序是：

1. 先用血檢、尿檢、血壓確認狀態。
2. 和獸醫討論是否需要腎臟處方飲食。
3. 逐步換食，避免突然全換造成拒食。
4. 定期追蹤體重、磷、鉀、血壓和尿蛋白。

若貓完全不吃處方糧，請回診討論替代方案。對腎病貓來說，願意穩定進食本身就很重要。

## 居家照護 6 件事

### 1. 每週量體重

體重是最容易在家追蹤的指標。用同一台秤、同一時間量，連續下降比單次數字更值得注意。

### 2. 記錄喝水和尿塊

不用追求精密到毫升，至少注意水碗下降速度、尿塊大小、貓砂清理頻率是否改變。

### 3. 提高水分攝取

濕食、食物加水、多放水碗、使用貓喜歡的碗或飲水機，都能幫忙。不要強灌水。

### 4. 避免自行用藥

人用止痛藥、消炎藥對貓可能很危險。腎病貓用任何藥或保健品，都應先問獸醫。

### 5. 控制壓力

多貓衝突、搬家、砂盆位置改變，都可能讓腎病貓吃得更少、喝得更少。穩定環境也是照護的一部分。

### 6. 按時回診

早期可能數月追蹤一次；中後期或指標不穩時，頻率會更密集。追蹤不是形式，是為了及早調整治療。

## 什麼情況要盡快就醫？

- 24 小時幾乎不吃
- 持續嘔吐或腹瀉
- 明顯虛弱、躲藏、不願移動
- 尿量突然大幅改變
- 呼吸急促或牙齦蒼白
- 體重短期明顯下降

---

慢性腎病通常無法靠單一方法「治好」，但早期發現、穩定飲食、控制血壓與定期追蹤，能讓很多貓維持更好的生活品質。你可以用[症狀檢查器](/tools/symptom-checker)整理觀察，但診斷和分期仍要交給獸醫師。
