# 承富毛利試算框架

## 何時用

- Go/No-Go 評估預算合理性時
- 決定報價金額
- 結案後回算實際毛利
- 月結 / 季結利潤分析

## 承富核心公式

```
毛利 = 總收入 - 總直接成本
毛利率 = 毛利 / 總收入

純利 = 毛利 - 間接成本(辦公室、行政人員、軟體、稅)
```

## 成本結構拆解

### A. 直接成本(佔總收入估算)

| 類別 | 比例 | 細項 |
|---|---|---|
| **人力** | 35% | PM、執行、設計、外包 |
| **場地** | 20% | 租金、押金、清潔、水電 |
| **設備 / 佈置** | 18% | 舞台、音響、燈光、佈置、印刷 |
| **其他** | 7% | 餐飲、交通、禮品、公關、雜支 |
| **總計** | **80%** | |

### B. 間接成本(公司層面)
承富內部每月分攤:
- 辦公室租金:NT$ 60,000 / 月
- 行政人員:NT$ 50,000 / 月
- 軟體訂閱:NT$ 15,000 / 月(含 AI 系統)
- 會計 / 法務:NT$ 20,000 / 月
- 其他:NT$ 15,000 / 月
- **合計:NT$ 160,000 / 月**

若承富月營業額 NT$ 300 萬 → 間接成本佔 5.3%

### C. 目標毛利率

- **優質標案**:25-30%(結構合理,執行順利)
- **標準標案**:18-22%(承富基本線)
- **艱困標案**:10-15%(邊際可做)
- **虧損標案**:< 5%(拒接或策略性考慮)

## 試算工作表

### 輸入
```
案名:2026 海洋廢棄物整治案
總收入(合約金額未稅):NT$ 3,800,000
期程:6 個月
```

### 人力試算

| 角色 | 工時 | 費率 | 小計 |
|---|---|---|---|
| 資深 PM | 300 | 2,500 | 750,000 |
| 設計師 | 100 | 2,000 | 200,000 |
| 執行 × 2 | 200 | 1,200 | 240,000 |
| 總監督導 | 40 | 4,000 | 160,000 |
| 臨時工作人員 | 150 | 800 | 120,000 |
| **人力小計** | | | **1,470,000** |

### 場地試算
- 6 場活動 × NT$ 80,000 = 480,000
- 場地相關水電 / 清潔 / 押金:NT$ 120,000
- **場地小計:NT$ 600,000**

### 設備 / 佈置試算
- 舞台 / 音響 / 燈光(6 場)= 420,000
- 背板 / 海報印刷 = 80,000
- 直播設備(3 場)= 120,000
- 花材 / 佈置 = 60,000
- **設備小計:NT$ 680,000**

### 其他試算
- 餐飲(6 場 × 80 人 × NT$ 350 /人)= 168,000
- 交通 / 油資 = 30,000
- 禮品 / 紀念品 = 50,000
- 備用金 = 80,000
- **其他小計:NT$ 328,000**

### 直接成本總計

```
人力:1,470,000  (38.7%)
場地:  600,000  (15.8%)
設備:  680,000  (17.9%)
其他:  328,000  (8.6%)
總計:3,078,000  (81.0%)
```

### 毛利計算

```
收入:NT$ 3,800,000
- 直接成本:NT$ 3,078,000
= 毛利:NT$ 722,000
= 毛利率:19.0%
```

→ 符合承富目標毛利率(18-22%),**可執行**。

## 毛利不足時的調整

### 方法 1:議價增收
- 與客戶協商增項(加場、加人數)
- 提「超額服務」換取預算調升(5-10%)

### 方法 2:節省人力
- 主 PM 降級(資深 → 中級),但要能控品質
- 減執行人員數(1 人多工,但累)
- 外包部分(有時外包比自家人便宜)

### 方法 3:場地議價
- 用承富長期合作場地(折扣固定)
- 選非熱門時段(週一至週四、早上)
- 多場綁單(例:簽下全年 4 場可拿大折扣)

### 方法 4:設備優化
- 多場共用設備(一次搬運費、多場用)
- 承富自家設備庫存(音響、簡單燈光)優先

### 方法 5:改變預期
- 接受較低毛利(10-15%)換策略標竿案
- 補破洞用別的高毛利案

## 隱性成本(易漏算)

- **超時加班**:執行期後 20% 時間被佔用但沒算
- **客戶變更**:改第 N 次要收費,但常沒收到
- **應收帳款風險**:政府結案付款可能拖 60-90 天,現金流壓力
- **人員流動**:PM 離職的交接成本
- **設備折舊**:自家設備每年折舊
- **承富品牌投入**:不直接算到案,但實際有成本

## 風險提示

### 紅旗(絕不接)
- 人力費 > 總預算 50% 但期程 > 6 個月 → 人累垮
- 設備自掏腰包(客戶不付)> 總預算 10%
- 要求「無限次修改到滿意」(沒有上限)

### 黃旗(需條款保護)
- 合約無「變更收費」條款 → 加入
- 應收帳款 > 3 個月才支付 → 要求期中款
- 客戶有拖付歷史 → 要求保證人

## 月度毛利回顧

承富每月末:
1. 列本月執行案毛利
2. 比較預期 vs 實際
3. 差異分析(哪裡超支?)
4. 下月調整策略
