# 承富活動預算分配比例

## 何時用

- 收到客戶總預算,需分配到細項
- 寫服務建議書第五章(預算明細)
- 評估預算是否足夠執行

## 承富四大塊分配原則

### 標準分配(記者會 / 發表會類)

| 類別 | 比例 | 內容 |
|---|---|---|
| **人力** | **35%** | PM 工時、執行人員、主持、設計、攝影、直播團隊 |
| **場地** | **25%** | 租金、設備使用、基本水電、清潔 |
| **設備 / 佈置** | **20%** | 舞台、音響、燈光、視訊、背板、花材 |
| **其他 / 備案** | **20%** | 餐飲、交通、印刷、禮品、公關、雜支、備用金 |

### 變體 1:晚宴 / 頒獎類

| 類別 | 比例 | 備註 |
|---|---|---|
| 人力 | 30% | 相對少 |
| 場地 | 20% | 視場地類型 |
| 設備 / 佈置 | 25% | 更多美學 |
| **餐飲** | **15%** | 獨立列 |
| 其他 | 10% | |

### 變體 2:論壇 / 工作坊類

| 類別 | 比例 | 備註 |
|---|---|---|
| **人力** | **45%** | 主持、講者、協調大幅佔比 |
| 場地 | 20% | |
| 設備 | 15% | 簡單 |
| 其他(餐飲、講師費) | 20% | |

### 變體 3:展覽 / 體驗類

| 類別 | 比例 | 備註 |
|---|---|---|
| **場地 / 裝潢** | **40%** | 最大佔比 |
| 人力 | 25% | |
| 設備 | 15% | |
| 其他 | 20% | |

## 預算級距與品質對照

### 低預算(NT$ 50 萬 - 200 萬)
- **限制**:難有大型舞台、無直播、無明星講者
- **策略**:精簡辦、強化在地合作(共創感)
- **風險**:品質把控要嚴,一次出錯就糟

### 中預算(NT$ 200 萬 - 600 萬)
- **大宗**:承富日常標案區間
- **包含**:標準記者會、主持、一般佈置、基本直播
- **策略**:穩健執行,品質可預期

### 高預算(NT$ 600 萬 - 2,000 萬)
- **能做**:大型晚宴、國際級製作、明星講者、多媒體整合
- **策略**:視覺與體驗投資,記憶點鮮明
- **風險**:供應商選擇多,PM 管理能力要強

### 極高預算(> NT$ 2,000 萬)
- **層級**:政府大型論壇、兩岸 / 國際盛會、企業年度大會
- **承富經驗**:需合作更大型的供應商,承富作為策劃主體
- **風險**:驗收標準高,交付物多

## 人力工時試算

### PM 工時估算

| 活動規模 | PM 投入工時 |
|---|---|
| 小型(< 50 人) | 80-120 hr(約 2 週) |
| 中型(50-150 人) | 150-250 hr(約 4-6 週) |
| 大型(150-500 人) | 300-500 hr(約 2-3 月) |
| 超大(500+ 人) | 600+ hr(多 PM 分工) |

### 承富工時費率

- PM(承富內部) · NT$ 1,500 / hr(基本)
- 資深 PM · NT$ 2,500 / hr
- 總監 · NT$ 4,000 / hr
- 設計師 · NT$ 2,000 / hr
- 執行人員 · NT$ 800-1,200 / hr

### 範例:NT$ 300 萬活動工時預算分配

假設人力 35% = NT$ 105 萬
- 專案 PM 200 hr × 2,500 = 50 萬
- 設計 60 hr × 2,000 = 12 萬
- 執行 3 人 × 40 hr × 1,200 = 14.4 萬
- 承富總監督導 20 hr × 4,000 = 8 萬
- 現場工作人員 × 3 人 × 8 hr × 800 = 1.92 萬
- 備用 18.68 萬

## 設備佈置比例細分

以佈置 20% 來看(假設 NT$ 300 萬活動 = NT$ 60 萬):
- 主舞台與背板:40%(24 萬)
- 音響:20%(12 萬)
- 燈光:15%(9 萬)
- 直播 / 攝影:15%(9 萬)
- 花材 / 佈置細節:10%(6 萬)

## 餐飲試算

### 茶點 / 冷餐
- NT$ 200-400 / 人(單純飲料 + 點心)

### 午餐便當 / 簡餐
- NT$ 300-500 / 人

### 正式晚宴
- NT$ 1,200-2,500 / 人(10 人一桌,NT$ 12,000-25,000 / 桌)

### VIP 級晚宴
- NT$ 3,000-5,000 / 人

## 預算警訊與拒絕線

### 警訊(需與客戶討論)
- 總預算 < 市場行情 70% → 品質會下降,需取捨
- 人力比 < 25% → 執行力堪憂
- 場地比 > 50% → 其他項目被犧牲

### 拒絕線(No-Go 配合 Skill 02)
- 總預算 < 市場行情 50%(做了虧錢)
- 期程短於合理範圍(品質難保)
- 要求「最低價保證最好品質」(不可能)

## 議價空間

承富慣用:
- **人力費**:可議 10-15%(資深 PM 降到中級 PM 執行)
- **場地**:看關係,15-25% 可議
- **設備**:承富有長期合作廠商,15-20% 折扣常見
- **其他**:可精簡(餐飲升級/降級、禮品選配)

## 超額承諾策略

承富建議書常加「超額承諾」(不列入預算):
- 結案報告(客戶關心)
- 3 個月成效追蹤
- 6 個月後的回顧座談會
- 媒體露出彙整

## 輸出格式

```markdown
# [活動名] · 預算規劃

## 總預算
NT$ 3,000,000(未稅)

## 類別分配

| 類別 | 比例 | 金額 | 說明 |
|---|---|---|---|
| 人力 | 35% | NT$ 1,050,000 | PM 200 hr + 設計 60 hr + 執行 3 人 |
| 場地 | 25% | NT$ 750,000 | XX 會議廳含基本設備 |
| 設備 / 佈置 | 20% | NT$ 600,000 | 舞台 + 音響 + 燈光 + 直播 |
| 其他 / 備案 | 20% | NT$ 600,000 | 餐飲 + 印刷 + 備用金 |

## 細項表(詳細)
[...逐項列出...]

## 風險管理
- 保留 5% 備用金(NT$ 150,000)
- 若超支:優先縮減餐飲/禮品
- 絕不縮減:人力、主舞台、直播

## 超額承諾(不入預算)
- 結案報告(含媒體分析)
- 3 個月成效追蹤
```

## 自我檢查

- [ ] 四大塊總和 = 100%
- [ ] 比例符合該類活動慣例
- [ ] 人力 ≥ 25%
- [ ] 有備用金 5-10%
- [ ] 每項有明確對應物 / 工時
- [ ] 金額格式 NT$ XX,XXX
- [ ] 含稅 / 未稅標註清楚
