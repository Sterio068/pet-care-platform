# 承富服務建議書 5 章模板

## 何時用

- Go/No-Go 決定 Go 之後,3-7 天內要交完整建議書
- 政府標案最常見結構
- 企業提案可簡化(3-4 章)

## 五章標準結構

### 第一章:需求理解與目標設定(8-12%)

**目的**:證明承富懂客戶痛點,不是「我有什麼賣給你」,是「我懂你要什麼」。

**子節**:
1.1 客戶背景與當前挑戰
1.2 專案核心目標(引用招標須知原文 + 承富解讀)
1.3 成功關鍵指標(KPI / 量化指標)
1.4 與承富核心能力的契合點

**承富技巧**:
- 一定要**引用招標須知原文**(用楷體或引號),後面寫「承富的理解」。這動作讓評審知道你讀過全文。
- 不要盲目捧客戶(老闆很有眼光之類),用事實陳述。

### 第二章:執行策略(25-30%)

**目的**:給方法論框架,證明承富有「結構化思考能力」。

**子節**:
2.1 策略核心(1-3 個支柱)
2.2 階段劃分(前期 / 執行 / 收尾)
2.3 主要方法(承富如何做這件事)
2.4 **在地整合**(承富殺手鐧 · 政府評審愛看)
2.5 數位 / 線上整合(近年必備)

**承富的「在地整合」**:政府評審特別重視。承富每份建議書都要提:
- 在地媒體資源(地方記者、社區電台、里長系統)
- 在地 KOL(在地美食部落客、地方意見領袖)
- 實體據點(公部門空間、圖書館、社區中心)
- 在地廠商(在地工作室、印刷、物流)

### 第三章:執行計畫(30-35%)

**目的**:細節、細節、細節。評審看這章判斷「這公司真的會做,還是嘴炮」。

**子節**:
3.1 工作分解結構(WBS)
3.2 詳細時程表(甘特圖)
3.3 各工作項的交付物與驗收標準
3.4 具體活動/產出(例:記者會 3 場、媒體投放 12 篇)
3.5 變更管理與風險應對
3.6 溝通機制(每週/月報告、LINE 群、專案平台)

**承富技巧**:
- 甘特圖用具體週期(不是「第一階段」)
- 每個交付物寫「量化標準」(字數、頁數、人次、露出量)
- 寫「若出現 X 狀況,承富會 Y」的備案(顯示成熟度)

### 第四章:團隊配置與管理(15-20%)

**目的**:讓評審安心把案子交給承富這群人。

**子節**:
4.1 專案組織圖(PM / 執行 / 支援)
4.2 主要成員簡歷(每人 100-150 字 + 近 3 年實績)
4.3 投入時數分配(每人 / 每月)
4.4 品質管理機制
4.5 承富公司簡介(近 3 年實績、獎項、合作媒體)

**承富技巧**:
- 簡歷寫**具體成果**,不只職稱(「曾執行 XX 活動,達成 X 曝光」)
- 投入時數要合理(超過 80 小時 / 人月就太多,評審會質疑可行性)
- 公司簡介放文末,不喧賓奪主

### 第五章:預算與成果承諾(15-20%)

**目的**:清楚說明錢怎麼花、會有什麼回報。

**子節**:
5.1 預算明細(人力 / 材料 / 執行 / 管理 / 其他)
5.2 成本合理性說明(為什麼這樣分)
5.3 預期成果(量化 + 質化)
5.4 超額承諾(承富額外贈送的服務 / 報告)
5.5 驗收與保固

**承富技巧**:
- 預算**百分比要合理**(人力不低於 35%,若被壓到 25% 以下是警訊)
- 超額承諾列 2-3 項(例:贈送 1 場完整結案記者會、6 個月後成效追蹤報告)
- 驗收章節明確列出交付物清單

## 附錄(必備)

- A. 過往類似案例(3-5 個)
- B. 公司簡介 / 證書 / 獎項
- C. 合作媒體列表
- D. 風險管理計畫(詳細版)

## 頁數建議

- **政府標案**:40-60 頁(含附錄)
- **企業提案**:20-30 頁
- **超過 80 頁**:精簡,評審讀不完

## 視覺標準

- 字型:**思源宋體** 11-12pt(正體穩重)
- 標題:**思源黑體** 粗體
- 顏色:承富藍 #0F2340 + 灰階
- 配圖:**每 3-5 頁一張**(流程圖、組織圖、時程圖)
- 資訊圖表:簡潔、每張有標題與重點說明

## 承富過往優質範例(few-shot)

若有 knowledge-base/historical/ 中的**優質建議書**檔案:
- file_search 找「建議書」「服務建議書」
- 引用其結構與用字風格
- **不直接複製**(避免被視為套版)

## 自我檢查

撰寫完畢前:
- [ ] 5 章結構齊全
- [ ] 每章比例符合(需求 10% / 策略 28% / 執行 33% / 團隊 18% / 預算 18%)
- [ ] 有甘特圖、組織圖、預算表三張關鍵圖
- [ ] 引用招標須知原文至少 5 處
- [ ] 「在地整合」章節具體到可驗證(有名字、有地點)
- [ ] 所有承富用字(禁用詞 / 稱謂 / 格式)正確
- [ ] 頁數 30-60 頁
- [ ] 附錄完整
