# 承富舞台與動線設計原則

## 何時用

- 活動場地確認後(踏勘完)
- 進入執行規劃階段
- 需要給客戶看現場配置圖

## 舞台位置三原則

### 1. 視線原則
- 所有觀眾都能看到舞台主要區域
- 柱子、樑不在觀眾與舞台之間
- **T 形舞台**:展演延伸走道,拉近觀眾距離

### 2. 動線原則
- 講者 / 藝人從**側邊**上舞台(不穿過觀眾)
- 舞台**後方有 2 公尺**以上後場準備區
- 舞台與最近出口距離 < 10 公尺(安全)

### 3. 背板原則
- 背板**高度 ≥ 3m**(鏡頭內不穿幫)
- 背板**寬度 ≥ 舞台寬**(構圖完整)
- 承富 logo / 客戶 logo 位置在**左上**(華人視覺焦點)

## 四種活動類型的標準配置

### A. 發表會 / 記者會

```
      媒體攝影區 ┐
                 │
 ┌─────────────┐ │
 │   主背板     │ │
 │  (3m × 6m)  │ │
 │            │ │
 │  [講台]    │ │ ◀── 側門(講者進出)
 │            │ │
 └─────────────┘ │
                 │
 ┌─────────────┐
 │   媒體座位 (前 3 排) │
 ├─────────────┤
 │   一般來賓  │
 └─────────────┘
     ↑
    入口
```

**關鍵**:
- 媒體攝影區在**觀眾席後**(全景鏡頭)
- 電視攝影機 tripod 站位預留
- 講者側門 = 不穿過觀眾區

### B. 晚宴 / 頒獎

```
 ┌─────────────────────────┐
 │        主舞台            │
 │   [頒獎區] [演出區]      │
 └─────────────────────────┘

 ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐
 │VIP│ │ 1 │ │ 2 │ │ 3 │ │VIP│ ← 圓桌區
 └──┘ └──┘ └──┘ └──┘ └──┘

 ┌──┐ ┌──┐ ┌──┐ ┌──┐
 │ 4 │ │ 5 │ │ 6 │ │ 7 │
 └──┘ └──┘ └──┘ └──┘

 ←── 廚房 / 餐飲動線 ──→
```

**關鍵**:
- VIP 桌 = 視線最好位置(不一定最前,但最中)
- 圓桌 10 人制 / 方桌 6 人制
- 服務動線**不穿越表演區**

### C. 工作坊 / 論壇

```
 ┌──────────────┐
 │  主持人席    │ ← 側邊
 └──────────────┘

 ┌────────┐ ┌────────┐
 │ 投影區  │ │ 討論區  │
 └────────┘ └────────┘

 ┌──┐ ┌──┐ ┌──┐
 │組 │ │組 │ │組 │ ← 分組桌
 │ 1 │ │ 2 │ │ 3 │
 └──┘ └──┘ └──┘
```

**關鍵**:
- 多中心設計(非單一舞台)
- 分組桌便於討論
- 主持人可快速到各組

### D. 展演 / 產品體驗

```
 ┌──────────────┐
 │   主要展區    │
 │  [產品展示]  │
 └──────────────┘

 ┌────┐ ┌────┐
 │體驗1│ │體驗2│ ← 分區體驗站
 └────┘ └────┘

 入口 →→→ [引導動線] →→→ 出口
```

**關鍵**:
- **單向動線**(避免人潮對沖)
- 體驗站間距 ≥ 3m(不擁擠)
- 出口 ≠ 入口

## 動線設計的 3 個禁忌

### 禁忌 1:人潮對沖
- ❌ 主要入口與出口同一個
- ❌ 餐飲區動線與主要活動區交叉
- ✅ 單向環形動線

### 禁忌 2:講者穿過觀眾
- ❌ 講者從觀眾席走道上台
- ❌ 後台入口在觀眾席後方(需穿過)
- ✅ 側邊 / 後方專用通道

### 禁忌 3:媒體影響活動
- ❌ 攝影記者站在觀眾前(擋視線)
- ❌ 直播設備擋緊急出口
- ✅ 專設媒體區(後方或側邊平台)

## 承富慣用尺寸

### 舞台
- 記者會:**3m × 6m**(3 深 × 6 寬)
- 晚宴主舞台:**5m × 10m**
- 頒獎主舞台:**4m × 12m**(有側翼)
- 工作坊:**2m × 4m**(簡約)

### 觀眾區
- 座椅間距:**90 cm**(前後)+ **60 cm**(左右)
- 走道:**1.2 m** 主走道 + **90 cm** 次走道
- 圓桌直徑:**180 cm**(10 人)
- 方桌:**180 × 90 cm**(6 人)

### 後場
- 講者休息室:**至少 15 ㎡**
- 工作人員區:**至少 20 ㎡**
- 器材倉:**10 ㎡**

## 視覺重點配置

### 客戶 / 承富 logo 位置
- 主背板左上(華人閱讀從左到右)
- 講台正面(電視畫面構圖中)
- 背板右下(廣角鏡頭會帶到)

### 直播 / 拍攝考量
- 主機位:觀眾席後方中央
- 副機位:兩側 45 度
- 特寫機位:舞台側邊(近)
- 逃避:**避免逆光**(窗戶、燈在後)

## 現場工作人員站位

- **總 PM**:中控台(能看到全場)
- **主持引導**:入口
- **機動**:人潮瓶頸點
- **安全**:緊急出口(各一名)

```markdown
# [活動名] · 現場配置規劃

## 場地資訊
- 場地:XX 會議廳
- 實際容量:150 人
- 活動人數預估:120 人

## 配置類型
記者會(A 型)

## 平面圖
[附圖]

## 關鍵配置
- 主背板:3m × 6m · 位置東側
- 講者動線:東側側門 → 舞台(全程不經觀眾)
- 媒體區:舞台正前方 8m 處(架高平台)
- 觀眾座椅:80 張(10 排 × 8 張)

## 動線規劃
- 入口 A(主):西北門
- 入口 B(媒體):北門
- 緊急出口:東南、西南(標示)
- VIP 專用通道:西門

## 設備需求清單
- 舞台組:1 套
- 音響:主 PA + 監聽 × 2
- 燈光:主燈 × 4 + 側燈 × 4
- 直播:3 機位 + 導播台
- ...

## 時序表(現場工作流程)
T-2h:工作人員進場
T-1h:彩排
T-0:活動開始
T+1h:結束,20 分鐘內復原
```

## 自我檢查

- [ ] 平面圖清楚標示所有區域
- [ ] 人潮動線無對沖
- [ ] 講者動線不穿越觀眾
- [ ] 媒體配置不影響活動
- [ ] Logo 位置符合視覺焦點原則
- [ ] 安全出口標示明確
- [ ] 工作人員站位清楚
- [ ] 符合消防安全法規容納數
