# 社群貼文 · 承富 3 種 Hook 開場法

## 何時用

- Facebook / Instagram / LinkedIn 貼文
- 承富為客戶經營社群
- 承富自家品牌社群

## 核心原則

社群貼文**前 2 行決定觸及率**。演算法只會把「前面抓得住人的貼文」推給更多人。

承富慣用 3 種 hook(根據貼文目標選):

---

## Hook 1:提問式(適合互動 / 品牌連結)

**公式**:提一個貼近讀者生活的問題,讓他們想回答。

**範例**:
```
「上次有人丟垃圾到海裡 是什麼時候?」

答案可能出乎你意料。

環境部最新調查:我們每天的海洋塑膠垃圾量,
相當於每小時 3 台垃圾車倒進海。

今年承富協辦的海廢整治成果:
- 清理海岸 45 公里
- 回收塑膠 1,280 公噸
- 串聯 10 個在地社區

👇 留言告訴我你最近一次淨灘/減塑經驗?

#海洋廢棄物 #淨灘 #承富
```

**適用**:品牌感建立、喚起關注、凝聚社群

**禁忌**:
- ❌ 問封閉式問題(「你知道 X 嗎?」)
- ❌ 問太難的專業問題(沒人答得出)

---

## Hook 2:衝突式(適合新聞性 / 話題性)

**公式**:把常識與真相對比,製造「原來如此」的驚奇感。

**範例**:
```
我們以為回收率高 = 環保做得好。

但真相是:臺灣每年 850 萬噸垃圾,只有 25% 真正被
重新利用,其他「回收」的最後還是焚燒或掩埋。

環境部本週公布的減量路線圖,首次把這個數字攤開。

承富負責策劃的成果發表會,讓這些數字上新聞版面。

想知道新的減廢目標是什麼嗎?完整影片已上傳官網👇

#減廢 #循環經濟 #承富
```

**適用**:公告政策、揭露議題、有數據佐證的主題

**禁忌**:
- ❌ 製造假衝突(客戶會失望)
- ❌ 過度負面(讀者會略過)
- ✅ 有反轉,但正向收尾

---

## Hook 3:數字式(適合成果 / 新品 / 效益)

**公式**:開場放一個具體數字,引發好奇。

**範例**:
```
3,500 位民眾 · 6 場活動 · 45 公里海岸線

這是承富與環境部本季一起寫下的成績單。

接下來 3 個月:
✨ 在地共創工作坊(10 個鄉鎮)
🌊 淨灘志工招募(500 個名額)
📣 校園巡迴演講(20 場)

報名從 4/20 開始,連結在 IG 首頁。

#共創 #在地行動 #承富
```

**適用**:活動成果、發表會配套、招募 / 促銷

**禁忌**:
- ❌ 數字不夠「具體」(「很多人」「大量」)
- ❌ 不同單位混用(要統一:人 / 場 / 次)
- ✅ 最多 3-5 個數字,太多讀不完

---

## 三種 hook 選擇決策樹

```
貼文目的是什麼?
├─ 建立品牌連結 / 日常內容 → Hook 1(提問式)
├─ 公告重要資訊 / 話題性 → Hook 2(衝突式)
└─ 成果 / 活動 / 招募 → Hook 3(數字式)
```

## 結構模板

```
[Hook 開場 · 前 2 行 · 約 50 字]
[延伸 / 說明 · 2-3 段 · 每段 1-2 行]
[CTA · 呼籲行動 · 1 行]
[Hashtag · 3-5 個]
```

## 平台差異

### Facebook
- 長度:200-400 字 OK
- 分段明確(前後段空一行)
- 鼓勵互動(提問 / 投票)
- 重要資訊在前段

### Instagram
- 長度:150-250 字(文字閱讀率低於 FB)
- 視覺為主,文字輔助
- 首 2-3 行特別重要(折疊前)
- Hashtag 可放 5-15 個(IG 友善)

### LinkedIn
- 長度:200-500 字(專業讀者耐心較好)
- 語氣偏專業(少表情符號)
- 結尾可放「深入討論」邀請
- Hashtag 3-5 個專業類

### Threads
- 類似 Twitter,簡短(≤ 200 字)
- 系列串連(主貼文 + 2-3 個 reply 展開)
- 語氣輕鬆

## 承富 hashtag 策略

### 三層 hashtag
1. **品牌層**:#承富創意整合行銷 或 #客戶品牌
2. **主題層**:#活動主題(例 #海洋廢棄物)
3. **通用層**:#臺灣 #公關行銷 #永續(被搜尋率高)

### 禁忌
- ❌ 堆 30 個 hashtag(看起來很 spam)
- ❌ 用英文 hashtag(承富客戶多為台灣本地)
- ❌ 用 #開箱 #業配 等商業類(破壞專業感)

## 承富 emoji 使用原則

- 每則 **最多 3-5 個** emoji
- 使用位置:標題前(吸睛)、列表(讓眼睛分段)、結尾(友善)
- **禁用**:😂 🤣 🥳 🎉(過度熱情)
- **推薦**:✨ 🌊 🌱 📣 💡 ✅(乾淨正向)

## 承富過往高觸及率貼文模式(參考)

### 模式 1:「你知道嗎」系列(提問式)
觸及率高,但需有專業內容支撐。

### 模式 2:幕後花絮(人性化)
記者會籌備過程、團隊討論片段。真實感 > 完美感。

### 模式 3:活動倒數(數字式)
「活動還有 7 天」「報名剩 50 個名額」等倒數,激發行動。

## 自我檢查

- [ ] Hook 在前 2 行抓住人
- [ ] 符合選定的 hook 類型(提問 / 衝突 / 數字)
- [ ] 無承富禁用詞
- [ ] Hashtag 3-5 個,三層兼顧
- [ ] Emoji 不超過 5 個
- [ ] 長度符合平台特性
- [ ] 有 CTA(呼籲行動)
- [ ] 圖文配合想法(文案若配圖要附建議圖稿方向)
