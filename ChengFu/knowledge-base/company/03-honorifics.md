# 承富稱謂規則

對不同對象使用不同稱謂,是承富專業度的基本展現。**對政府客戶錯一次,可能失分。**

## 一、對政府機關

### 1.1 機關名稱(全銜)

| 層級 | 稱謂模式 | 範例 |
|---|---|---|
| 中央 | 部 / 署 / 局 / 處 / 館 / 會 全銜 | 「行政院環境部」「國家通訊傳播委員會」「文化部文化資產局」 |
| 地方 | 直轄市政府 XX 局 / 處 | 「臺北市政府文化局」「新北市政府觀光旅遊局」 |
| 二線機關 | 主管機關 + 附屬全銜 | 「教育部體育署」「內政部移民署」 |

**重要**:
- 正式文書**全銜優先**(第一次提到用全銜,後面可簡稱)
- 簡稱必須正確:「文化部」不是「文部」;「環保局」是「環境保護局」;「觀光局」已改名為「觀光署」(2023/9)
- 遇到不確定機關名 → **查 https://www.ey.gov.tw/ 官方網站確認**,不要臆測

### 1.2 職稱

| 層級 | 稱謂 | 範例 |
|---|---|---|
| 首長 | 部長 / 署長 / 局長 / 處長 / 館長 / 主委 / 市長 / 縣長 | 「黃局長」「陳部長」 |
| 副首長 | 副部長 / 副局長 / 副市長 | 「李副局長」 |
| 簡任級 | 主任秘書 / 參事 / 主任 | 「王主秘」 |
| 薦任級 | 科長 / 專員 | 「張科長」 |
| 公務同仁 | 先生 / 小姐 / 女士 (保守)or 姓 + 職稱 | 「黃小姐」「李先生」 |

### 1.3 書信稱謂公式

**收文**:
- 尊稱 → 「鈞長」(上級)、「貴局」(平行)、「大鑒」(一般)
- 敬啟 → 「敬啟」「鈞鑒」

**自稱**:
- 對上級 → 「承富謹啟」「敝公司」
- 對平行 → 「承富」「我司」

**結尾**:
- 「此致」「敬請 鑒核」「耑此」「敬頌 公祺」

## 二、對企業客戶

### 2.1 公司名(全稱 / 簡稱)
第一次書寫 **公司全名 + 股份有限公司**,後面可簡稱。
- 「遠傳電信股份有限公司(以下簡稱『遠傳』)」

### 2.2 職稱

| 職級 | 稱謂 |
|---|---|
| C-level | 董事長、執行長、總經理、副總經理 |
| 副總 / 協理 | 副總、協理、經理 |
| 一般 | 經理、副理、主任、專員、先生/小姐 |

**規則**:
- **姓 + 職稱**(如「陳經理」),不要直呼名字(除非對方主動說可以)
- 不確定職稱 → 用「先生/小姐」加姓

## 三、對媒體

| 對象 | 稱謂 |
|---|---|
| 主筆、總編輯 | 「XX 主筆」「XX 總編」 |
| 編輯 / 組長 | 「XX 組長」「XX 編」 |
| 記者 | 「XX 記者」「XX 記者朋友」 |
| 主持人 | 「XX 主持人」 |

新聞稿抬頭通常寫「媒體聯絡人」+ 姓名電話 Email(承富固定格式)。

## 四、對合作廠商

| 類型 | 稱謂 |
|---|---|
| 設計公司 / 活動公司 | 「XX 公司」「XX 設計師」 |
| 工作室 | 「XX 工作室」 |
| SOHO / 個人 | 「XX 老師」(尊稱)or 「XX 先生/小姐」 |

承富立場:**合作,不是發包**。文案中用「夥伴」「合作」不用「廠商」「下屬」。

## 五、承富內部自稱

### 對外(書信、新聞稿、提案)
- 正式:「承富創意整合行銷有限公司(以下簡稱『承富』)」
- 簡稱:「承富」「我司」「本公司」
- **禁用**:「我們」「我方」「咱們」「這邊」

### 對內(同仁之間)
- 平輩、下對上:姓 + 職稱(「王 PM」「陳設計」「李組長」)
- 承富老闆 → 團隊內叫「老闆」或「X 總」
- Sterio(顧問)→ 視熟識度,「Sterio」或「Sterio 哥/姐」

## 六、AI 應用原則

### 主管家與所有對外寫作 Agent
每次產出前:
1. 辨識對象類型(政府 / 企業 / 媒體 / 廠商 / 內部)
2. 依表查對應稱謂
3. 不確定時**主動問使用者**:「這是要給 XX 的文件嗎?需要用哪種稱謂?」

### 稱謂範例庫(AI 可直接用)

**寄給政府局長**:
> 黃局長 鈞鑒:
> 頃悉貴局辦理 XXX 推廣案…
> 敬祝 公祺
> 承富創意整合行銷有限公司 敬啟

**寄給企業經理**:
> 陳經理 您好:
> 針對貴公司 XXX 專案,承富謹提供…
> 順頌 商祺
> 承富 王 PM

**寄給媒體記者**:
> 媒體朋友 您好:
> 承富謹代表客戶 XXX 發布新聞訊息如附…
> 敬祝 採訪順利
> 媒體聯絡人:王 PM · 02-XXXX-XXXX · pm@chengfu.com

## 七、失誤處理

若不小心用錯稱謂(客戶指正):
1. **立即私訊道歉**(不公開檢討)
2. 更新承富 CRM 系統(Agent 17)備註
3. 下次所有互動使用正確稱謂
