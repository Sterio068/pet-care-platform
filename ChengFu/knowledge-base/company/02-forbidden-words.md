# 承富禁用詞表

每次 AI 產出繁中文字,**務必替換**以下詞彙。對政府客戶尤其嚴格。

## 二、網路語 / 網紅梗(政府 / 企業皆避)

| 禁用 | 備註 |
|---|---|
| 爆款、必勝、超猛、神器 | 浮誇 |
| 咱們、搞定、齁 | 過度口語 |
| 哈哈、XD、lol | 不正式 |
| YYDS、絕絕子、蒸、狠活 | 網紅梗 |
| 躺平、擺爛、內卷 | 對政府敏感 |
| 人均、牛頭 | 大陸網紅用語 |

## 三、政治 / 宗教 / 族群敏感詞(承富主客戶是政府,極嚴)

避免使用可能觸發敏感的詞彙,改中性表達:
- 涉及兩岸的措辭:一律用官方文書慣用詞(「臺灣」「中國大陸」「海峽兩岸」)
- 涉及族群:中性描述(避免刻板印象)
- 涉及宗教:只描述事實(不評論教義)
- 涉及特定政黨:保持中立(承富對政黨中立)

## 四、法律 / 合規風險詞(謹慎使用)

| 詞 | 風險 | 替代 |
|---|---|---|
| 保證、一定、絕對 | 合約風險 | 「承富承諾以 XX 機制…」 |
| 獨家、唯一 | 可能不實 | 「承富獨到的…」 |
| 最便宜、最優 | 廣告不實 | 「承富具競爭力的…」 |
| 救命、包醫 | 醫療法 | 產業不涉及醫療時避免 |

## 五、承富內部特定禁用

承富歷史中曾被客戶糾正的詞(重要,避免重蹈):
- 「承富團隊」(被某客戶認為浮誇)→ 改「承富」
- 「頂級」「頂尖」「領先」(被視為自誇)→ 用具體事實描述
- 「一站式」「全方位」(太泛)→ 列出具體範疇

## AI 應用

主管家 Agent 與所有對外寫作 Agent 都會引入本檔。
每次產出都應做內部檢查,若發現禁用詞 → 替換 → 再輸出。

若使用者**刻意**要求使用大陸用語(例:寫給大陸客戶),可例外處理,但應提醒:
「這段將使用簡體中文用語。若是對台灣客戶請改用繁中正體。」
