# 承富品牌口吻(Brand Voice)

## 基本定位

承富是 **10 年政府標案經驗為核心**的公關整合行銷公司。
對外形象:**專業、正式、在地、可靠**,不追求網紅化。

## 四個面向

### 1. 對政府客戶
- 稱謂:**單位全銜 + 職稱**(如「環境保護局黃局長」不是「環保局黃」)
- 語氣:**敬語、公文體**(「敬請」「頃悉」「惠賜」「謹陳」)
- 避免:過度熱情、網路用語、表情符號(除非受邀)
- 範例:「承富敬啟:茲就貴局辦理之『XXX 推廣案』,謹提供服務規劃如下…」

### 2. 對企業客戶
- 稱謂:**公司 + 職稱 + 姓**(如「遠傳電信陳經理」)
- 語氣:**專業但稍放鬆**(仍不口語)
- 可以:「團隊精心策劃」「期盼為您打造」「與貴公司並肩」
- 避免:「我們」(承富自稱用「承富」或「我司」)

### 3. 對一般讀者(新聞稿 / 社群)
- 語氣:**說明清楚、重要資訊前置**
- 新聞稿:AP Style 倒三角(最重要事實在前)
- 社群:可用一個問題 / 數字 / 衝突點 hook 開場,但仍保持專業感
- 避免:網紅梗、浮誇字眼(「超猛」「必須」「爆款」)

### 4. 內部溝通(PM ↔ 設計 ↔ 業務)
- 語氣:**精簡、具體、有截止時間**
- 使用 bullet point 列清單
- 必提:責任人 + 截止日 + 交付物

## 承富「絕不說」的 5 句話

1. ❌「這個案子我們很有把握」 → ✅「承富評估成功率約 XX%」
2. ❌「別家公司做不到」 → ✅「承富的差異化在於…」
3. ❌「保證如期完成」 → ✅「承富承諾以 XX 機制控管進度」
4. ❌「客戶都很滿意」 → ✅「過往客戶回饋包含…(具體引述)」
5. ❌「這是業界潛規則」 → ✅ 不講(這種話不該出現)

## 三個必記要點

- **數字優先於形容詞**:「增加了 28%」比「大幅增加」有力
- **具體優先於抽象**:「包含 3 場記者會、12 篇媒體投放」比「全方位公關策略」可信
- **謙遜優先於吹噓**:「承富的執行經驗」比「業界頂尖團隊」得體
